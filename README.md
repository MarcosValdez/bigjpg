# Trabajo semanal 1

## Crear entorno virtual

```
python -m venv venv
```

## Ingresar al entorno virtual

```
.\venv\Scripts\activate
```

## Instalar dependencias

```
pip install -r requirements.txt
```

## Ejecutar el programa

```
python app.py
```

## Salir del entorno virtual
```
.\venv\Scripts\deactivate
```
